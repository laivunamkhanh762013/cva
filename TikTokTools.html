📖 Hướng Dẫn

🆓 CÁCH 1: Trực tiếp (Free, không cần API key)

1

Vào tab Dò Acc TikTok

Chọn phương thức "Trực tiếp (Free - không cần API)"

2

Nhập username & Tra Cứu

Tool sẽ dùng CORS proxy để lấy dữ liệu từ TikTok

3

Nếu bị lỗi

Vào API Settings → đổi CORS proxy (có 4 proxy để chọn)

🆓 CÁCH 2: TikTok oEmbed (Free, chính thức từ TikTok)

1

Chọn phương thức "TikTok oEmbed"

Tool gọi API chính thức của TikTok tiktok.com/oembed

2

Xem kết quả

Lấy được tên, avatar cơ bản. Nếu lỗi sẽ tự fallback về phương thức Trực tiếp

⚡ CÁCH 3: RapidAPI (Có gói Free)

1

Đăng ký RapidAPI Free

Vào rapidapi.com/hub → đăng ký miễn phí

2

Tìm "TikTok API" → Subscribe Free

Copy X-RapidAPI-Key từ Code Snippets

3

Cấu hình vào Tool

Vào API Settings → paste key → Save
